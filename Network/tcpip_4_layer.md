# 🌐 네트워크 - TCP/IP 4계층

## 📌 1. TCP/IP 4계층 개요
* **정의**: 인터넷에서 컴퓨터들이 데이터를 주고받을 때 사용하는 표준 통신 규약을 4개의 계층으로 단순화한 네트워크 모델
* **특징**: 이론적인 OSI 7계층과 달리, 실제 구현과 상용화에 초점을 맞춰 실무적으로 단순화된 구조

---

## 🔍 2. 계층별 상세 역할 및 특징



### 1. 응용 계층 (Application Layer)
* **주요 역할**: 사용자와 가장 밀접한 계층으로, 사용자가 네트워크 서비스를 이용할 수 있도록 인터페이스와 양식을 제공합니다. OSI 7계층의 **5, 6, 7계층(세션, 표현, 응용)**을 하나로 통합한 형태입니다.
* **데이터 단위**: 데이터 (Data)
* **주요 프로토콜**: `HTTP`(웹), `FTP`(파일 전송), `SMTP`(이메일), `DNS`(도메인 주소 변환)

### 2. 전송 계층 (Transport Layer)
* **주요 역할**: 응용 계층에서 받은 데이터를 세그먼트로 나누고, 송신자와 수신자의 프로세스(애플리케이션)를 포트(Port) 번호를 통해 연결하여 신뢰할 수 있는 데이터 통신을 담당합니다. OSI 4계층과 대응됩니다.
* **데이터 단위**: 세그먼트 (Segment)
* **주요 프로토콜**: `TCP`(신뢰성 보장, 연결형), `UDP`(속도, 비연결형)

### 3. 인터넷 계층 (Internet Layer)
* **주요 역할**: 데이터 패킷을 논리적 주소인 IP를 기반으로 목적지까지 가장 안전하고 빠른 경로를 통해 라우팅(경로 설정)하고 전송하는 계층입니다. OSI 3계층(네트워크 계층)과 대응됩니다.
* **데이터 단위**: 패킷 (Packet)
* **주요 프로토콜**: `IP`(주소 지정 및 경로 설정), `ICMP`(오류 보고), `ARP`(IP를 MAC 주소로 변환)

### 4. 네트워크 액세스 계층 (Network Access Layer)
* **주요 역할**: 물리적인 네트워크 매체(케이블, 무선 등)를 통해 기기 간에 데이터를 실제로 전달하는 계층입니다. OSI 7계층의 **1, 2계층(물리, 데이터 링크 계층)**에 해당하며, 데이터를 전기 신호로 변환하여 물리적인 네트워크 매체를 통해 기기 간에 실제로 전달합니다.
* **데이터 단위**: 프레임 (Frame) / 비트 (Bit)
* **주요 프로토콜 및 요소**: `이더넷(Ethernet)`, `Wi-Fi`, `MAC 주소`
